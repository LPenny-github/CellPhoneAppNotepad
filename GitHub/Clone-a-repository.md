# Clone a repository


## 問題描述


在 Windows 裝置上複製一份別人的 repository 到自己的電腦。


## 執行


1. 找到該 repository

1. 按 `Code` ，複製該 repository 的 HTTPS

1. 打開 PowerShell 或 CLI ， 抵至目的地資料夾

1. `git clone 你複製的那段HTTPS` ：把 repository 複製一份到你的目的地資料夾


## 驗證


* 承前步驟， `code ..` ：用預設編輯器打開 repository
